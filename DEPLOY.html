TopHub 已就绪 🚀

这是一个静态站 + GitHub Actions 自动刷数据的组合。下面的步骤会帮你把它部署到 https://<你的用户名>.github.io/tophub。

前置条件

• 一个 GitHub 账号

• 电脑装了 Git（下载）

一键部署 3 步

1 在 GitHub 上建一个空仓库，名字叫 tophub（不要勾 README / .gitignore / license，留空）

→ https://github.com/new

2 在 PowerShell / Git Bash 里执行下面这一段（把 YOUR_USERNAME 替换成你的 GitHub 用户名）：

cd tophub
git init
git add .
git commit -m "init: TopHub v4 with auto-refresh"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/tophub.git
git push -u origin main

3 在 GitHub 仓库的 Settings → Pages → Source 选 GitHub Actions

→ 1-2 分钟后 https://YOUR_USERNAME.github.io/tophub 就能访问

可选：开启 LLM 自动解读

Settings → Secrets and variables → Actions → New repository secret，加：

OPENAI_API_KEY    sk-xxxxxxxxxxxxxxxx

之后每天 9 点（北京时间）会自动让 LLM 给项目写最新解读。

手机访问

Safari / Chrome 打开 → 分享 → "添加到主屏幕" → 全屏 App 体验

部署完成后，你的 TopHub 就每天自动更新，随时能看 ✨
